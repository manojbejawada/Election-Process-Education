# CivicEdu — India Election Assistant 🇮🇳

CivicEdu is a premium, interactive web application designed to educate Indian citizens about the democratic process, election procedures, and their voting rights. Built with a focus on accessibility and modern aesthetics, it serves as a comprehensive guide for both first-time and experienced voters.

![CivicEdu Banner](https://images.unsplash.com/photo-1594122230689-45899d9e6f69?auto=format&fit=crop&q=80&w=1200&h=400)

## ✨ Features

- **Interactive Election Timeline**: Navigate through the various phases of the Indian election process, from notification to results.
- **Step-by-Step Voting Guide**: A clear, visual guide on how to register and cast your vote at the polling station.
- **Civic Glossary**: Comprehensive definitions of essential election terms like MCC, EVM, VVPAT, and more.
- **Knowledge Quiz**: Test your understanding of the Indian democratic system with interactive quizzes.
- **Multilingual Support**: Integration with Google Translate offering content in English, Hindi, and Telugu.
- **AI-Powered Assistance**: (In development) Contextual AI help for election-related queries.
- **Secure Authentication**: Built-in user accounts and progress tracking powered by Firebase.

## 🛠️ Technology Stack

- **Frontend**: Semantic HTML5, Vanilla CSS3 (Custom Design System), Modern JavaScript (ES6+).
- **Backend Services**: Firebase Authentication, Firestore Database.
- **Typography**: Playfair Display (Serif) & DM Sans (Sans-serif).
- **Integrations**: Google Translate API, Google Fonts.

## 🚀 Getting Started

### Prerequisites

To run this project locally, you just need a modern web browser.

### Installation

1. **Clone the repository**:
   ```bash
   git clone https://github.com/manojbejawada/Election-Process-Education.git
   ```

2. **Navigate to the project directory**:
   ```bash
   cd Election-Process-Education
   ```

3. **Open the application**:
   Simply open `index.html` in your browser or use a local development server like Live Server (VS Code extension).

## 🎨 Design Philosophy

CivicEdu follows a **"Modern Heritage"** design aesthetic, combining traditional Indian colors (Saffron, Navy, Green) with a clean, Brutalist-inspired digital layout. It prioritizes readability, smooth transitions, and a premium feel to make civic education engaging.
